# 📚 Anotações de Aula - Java Básico

## 📍 Aula 27: Métodos com Parâmetros
* **Conceito Chave:** Escopo de Variáveis.
* Variáveis criadas dentro de um método são *locais* e só existem dentro dele.
* Usamos o `return` para enviar o resultado de um cálculo para fora do método.
